The Benchmark

SSC-Bench operationalizes Spatial Structural Competence through orthographic three-view projections (Front View, Left View, Top View), producing verifiable outputs via executable Python code.

Framework overview: Dual-path diagnostic paradigm for evaluating SSC through Image Reproduction (SSU) and Code-as-Reasoning Line Completion (SSR).

SSU: Structural Understanding

Image-to-Code Reproduction

Models must reproduce complete three-view orthographic drawings (FV, LV, TV) as executable Python code, demonstrating the ability to translate visual spatial structures into coherent symbolic representations.

Evaluation: Accuracy & F1 Score

SSR: Structural Reasoning

Code-Based Line Completion

Given ground-truth code for an incomplete drawing, models must synthesize three missing structural lines (rendered as red dashed lines) through pure logical inference and geometric constraint reasoning.

Evaluation: Completion Rate (CR)

Key Findings

Comprehensive evaluation of 13 state-of-the-art multimodal large language models reveals systematic weaknesses in spatial structural competence.

01

View-Dependent Instability

While some models achieve reasonable performance on Front and Left views, there is a significant performance drop on Top View synthesis, indicating unstable spatial understanding that lacks cross-view consistency.

SSU Track
02

Complete SSR Failure

All 13 models achieved 0.00% completion rate on the code-as-reasoning path, demonstrating a fundamental inability to perform logical synthesis of geometric relations, even when provided with ground-truth symbolic context.

Critical Finding
03

Dual-Path Diagnostic Value

The benchmark successfully isolates failure attribution: errors arise from both perceptual structural understanding and logical reasoning stages, rather than a single processing bottleneck.
